# ICSR Workspace

## Что находится в архиве

Это локальное приложение для работы с кейсами ICSR:

- просмотр и поиск кейсов;
- открытие и редактирование случая;
- проверка заполнения кейса;
- экспорт и импорт XML;
- локальный голосовой ввод и голосовые команды.

## Состав пакета

- `server.py` — серверная часть приложения;
- `run_server.bat` — быстрый запуск на Windows;
- `install_requirements.bat` — установка зависимостей Python;
- `requirements.txt` — список нужных библиотек;
- `data.db` — локальная база данных;
- `static/` — интерфейс приложения;
- `models/` — локальные модели распознавания речи;
- `nlu/` — конфигурация голосового интерпретатора.

## Требования

- Windows;
- Python 3.11+;
- доступ к локальному браузеру;
- микрофон — если нужен голосовой ввод.

## Быстрый запуск

1. Распаковать архив в отдельную папку.
2. При первом запуске выполнить `install_requirements.bat`.
3. Затем запустить `run_server.bat`.
4. Открыть в браузере адрес:

`http://127.0.0.1:8030`

## Голосовой режим

Базовый локальный голосовой ввод работает за счёт моделей в папке `models`.

Если нужен расширенный голосовой ассистент со свободными командами, дополнительно требуется:

1. установить [Ollama](https://ollama.com/);
2. выполнить команду:

```powershell
ollama pull qwen3:4b
```

Без `Ollama` приложение всё равно запускается и работает, но интеллектуальная интерпретация голосовых команд будет ограничена.

## Примечание

Если `run_server.bat` закрывается сразу, сначала установите зависимости через `install_requirements.bat` и убедитесь, что `python` доступен в `PATH`.
